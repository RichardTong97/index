# Sistema de Vigilancias Médicas

Mockup del Sistema de Vigilancias Médicas.

## Publicación en Netlify

1. Entrar a https://app.netlify.com/drop
2. Arrastrar esta carpeta.
3. La aplicación quedará publicada automáticamente.

## Publicación en GitHub Pages

1. Crear un repositorio.
2. Subir todos los archivos.
3. Ir a Settings → Pages.
4. Source → Deploy from Branch.
5. Branch → main.
6. Guardar.

La URL será:

https://TU_USUARIO.github.io/NOMBRE_REPOSITORIO/
